Constants

MAX_OUTER_LENGTH

The maximum length of a OnceVec is 2^{MAX_OUTER_LENGTH} - 1. The performance cost of increasing MAX_OUTER_LENGTH is relatively small, but [T; N] does not implement Default for N > 32, which we need for initialization. So let us stick with 32.
