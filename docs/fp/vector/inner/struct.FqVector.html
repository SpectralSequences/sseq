Struct FqVector

Source 
pub struct FqVector<F: Field> {
    pub(super) fq: F,
    pub(super) len: usize,
    pub(super) limbs: Vec<u64>,
}
Expand description

A vector over a finite field.

Interally, it packs entries of the vectors into limbs. However, this is an abstraction that must not leave the fp library.
