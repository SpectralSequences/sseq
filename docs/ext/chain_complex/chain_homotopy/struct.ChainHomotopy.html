Expand description

A chain homotopy from $f to g$, or equivalently a null-homotopy of $h = f - g$. A chain map is a priori a collection of free module homomorphisms. However, instead of providing FreeModuleHomomorphism objects, the user is expected to give a function that computes the value of $h$ on each generator.

The $s$ shift of the original chain map $f - g$.

shift_t: i32

The $t$ shift of the original chain map $f - g$.

map: F

A function that given (s, t, idx, result), adds (f - g)(x_{s, t, i}), to result.

lock: Mutex<()>homotopies: OnceVec<FreeModuleHomomorphism<T::Module>>

Homotopies, indexed by the filtration of the target of f - g.
